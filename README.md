# Git Commands

1. git init - Initializes a new local Git repository
2. git status - Shows current status
3. git add - Adds files to staging
4. git commit - Saves changes
5. git branch - Shows branches
6. git checkout - Switch branches
7. git merge - Merges branches
8. git pull - Gets latest changes
9. git push - Uploads changes
10. git log - Shows commit history
